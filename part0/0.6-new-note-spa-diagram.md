# Exercise 0.6: New note in Single page app diagram

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note right of browser: The user writes a new note and clicks Save
    Note right of browser: JavaScript prevents the default form submission
    Note right of browser: JavaScript creates the note, adds it to the list and redraws the page

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
    activate server
    Note right of server: The request contains the new note as JSON
    server-->>browser: HTTP 201 Created
    deactivate server

    Note right of browser: The browser stays on the same page
    Note right of browser: No additional HTTP requests are made
```
